Testing the Screen Magnifier

We have tested the screen magnifier tool of Mac OS on the Amazon website. The keyboard combinations for zoom in and zoom out are "command+option+plus" and to zoom out "command+option+minus" respectively. We can also toggle between regular mode and magnified mode using keyboard combination "command+option+8". While using the screen magnifier tool for searching and purchasing a product from Amazon, we found that it is quite helpful for visually impaired people. However, the horizontal scroll is not quite intuitive, we have to move the cursor to the edge of the display to move left or right. On the other hand, the vertical scroll works fine with swipe gestures and arrow keys. Another notable challenge is that finding specific textboxes or, buttons takes more time in magnified mode compared to the regular mode. We have also tested the high contrast mode which increased visibility however reduced the elegance of the UI and even faded out some texts.

Low Vision Mode

We have added a low vision mode in our dialer application which includes magnification of the content and alteration of contrast. In low vision mode, all the contents and texts are magnified to 150%. Besides, a high contrast black and white theme is applied for background, foreground, and, texts along with yellow hover effects. Users can easily toggle between the low vision mode and the regular mode using the dedicated button located on the top of the page. We have used a single index.html file where switching between modes is done by changing the CSS file through Javascript.
